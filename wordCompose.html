放題（ほうだい）

用法:名詞/動詞ます形/形容動詞語幹/たい + 放題 = 表示恣意地從事某行為

例:食べ放題(吃到飽)、10分かけ放題(前10分鐘內通話免費)、言いたい放題(自顧自地說)

強（きょう）/弱（じゃく）

用法:數量詞 + 強/弱 = 表示比某數量稍微多/少一些 (去掉尾數時使用)

例:8割強(8成多左右)、2年弱(將近2年)

*注意:此用法較為曖昧，根據個人感覺、年代別對「稍微」的範圍認知可能會有所差異，使用上時需要小心。

加減（かげん）

用法:名詞/動詞ます形 + 加減 = 表示事物程度

例:焼き加減(煎烤的程度)、塩加減がいい(鹹度剛好)、火加減(火候)

扱い（あつかい）

用法:名詞(身分、作用、狀態) + 扱い = 當作…對待

例:子供扱い(當作小孩對待)、機密扱い(當作機密對待)、欠席扱い(當作缺席對待)

上がり（あがり）

用法:名詞/動詞ます形 + 上がり = 表示狀態剛結束沒多久

例:病み上がり(大病初癒)、雨上がり(雨過天青)、風呂上がり(剛出浴)

音痴（おんち）

用法:名詞 + 音痴 = 表示在某件事上遲鈍

例:運動音痴(運動白痴)、方向音痴(路痴)、機械音痴(機器白痴)

たて

用法:動詞ます形 + たて = 表示動作剛結束

例:焼きたて(剛烤好)、炊き立てご飯(剛煮好的飯)、出来立ての料理(剛做好的料理)

心地（ここち）

用法:動詞ます形 + 心地 = …的感覺

例:住み心地(住起來的感覺)、着心地(穿起來的感覺)、座り心地(坐起來的感覺)

*注意:當作接尾詞時發音為「ごこち」。
